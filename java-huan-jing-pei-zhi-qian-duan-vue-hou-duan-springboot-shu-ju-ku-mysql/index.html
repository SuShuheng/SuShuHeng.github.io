Ⅱ.环境安装保姆级教程

安装路径统一安装在非系统盘D盘，如果电脑没有分区只有系统盘C盘，则根据下方教程将安装根目录D:\更改为 C:\

（一）、安装流程

1.Java JDK 安装
  • 双击打开下载好的 jdk-25_windows-x64_bin.exe 安装包执行文件（见下图）

  • 安装程序页面点击下一步（见下图）

  • 点击更改，修改安装路径（见下图）

  • 在选择安装路径选择非系统盘D:\,然后创建一个新的文件夹WorkEnv (注意大小写规范!)并双击选择该文件夹后再创建一个新的文件夹 Java 并选择

  • 确保选择的安装路径是 D:\WorkEnv\Java\ 后点击 确定最后点击 下一步进行安装，安装完成界面参考下图
  • 最后打开文件资源管理器，查看路径 D:\WorkEnv\Java下是否有如下图片展示的文件结构，如果有则说明安装成功
2.Maven 安装
  • 找到并解压名为 maven-mvnd-1.0.3-windows-amd64.zip 的压缩包，解压路径为 D:\WorkEnv\Maven

  • 解压后在路径 D:\WorkEnv\Maven\maven-mvnd-1.0.3-windows-amd64\maven-mvnd-1.0.3-windows-amd64中的文件结构如果与下图一致，则安装成功

3.Node Js 安装

  • 找到并打开Git安装包（如下图）

  • 点击 Next

  • 接受协议并且点击 Next

  • 点击 Change修改安装路径,在 D:\WorkEnv\路径创建新的文件夹 NodeJs并双击该文件夹选择为安装路径，确保安装的路径为 D:\WorkEnv\NodeJs 后点击 OK后在安装页面再次确定安装路径后点击 Next开始安装

  • 点击 Next

  • 再次点击 Next

  • 点击 Install

  • 点击 Finish 完成安装

  • 文件资源管理器打开路径 D:\WorkEnv\NodeJs检查文件结构是否与下图一致，若一致则完成安装流程

4.Git 安装
  • 找到 Git-2.51.1-64-bit.exe安装包并双击打开
  • 点击 Install,默认路径安装即可（我的默认安装为 D:\Git）

  • 取消 View Release Notes勾选，并点击 Finish完成安装

已经安装IDEA Ultimate并且订阅的同学可以跳过此步骤

  • 找到 IDEA-24.0-破解版.zip压缩包并解压

  • 确保解压文件夹有如下结构

双击 ideaIU-2024.2.4.exe执行IDEA安装流程，并点击下一步

  • 点击 浏览 选择 磁盘D后点击 确定，安装软件会在 磁盘D自动创建目录如 D:\IntelliJ IDEA 2024.2.4,确定安装路径正确后点击 下一步

  • 勾选如图选项并点击下一步

  • 点击 安装 并等待安装完成

  • 安装结束后选择 否，我会在之后重新启动并点击完成

  • 在桌面找寻IDEA快捷方式并双击打开IDEA

  • 点击 下一个

  • 同意协议并且点击 继续

  • 点击 不发送

  • 出现下方页面后关闭 IDEA 并且在弹出窗口选择 退出

  • 打开刚刚打开的文件资源管理器，并进入文件夹 JetBrains全家桶激活(附带备用激活)

  • 进入文件夹 win2020-2024(一键激活)

  • 再次确定 IDEA是否已经被关闭，如果确定被关闭双击执行文件夹下的 IDEA激活.vbs

  • 在弹出的 "成功激活"页面点击确定后打开 IDEA软件

  • 同上方步骤一样，再次按步骤分别点击 下一个 、 确认 、 继续 后会出现下方界面

  • 点击左下角的齿轮按钮打开软件设置并选择 管理许可证..

  • 如果出现下方页面则说明安装并激活成功！

说明：Ultimate 版本 IDEA 在创建项目后端时会更方便快捷

  • 在打开的 VsCode页面中点击左侧扩展按钮，准备下载汉化插件以及JAVA插件

7.MySql安装

  • 找到并打开安装程序mysql-installer-community-8.0.44.0.msi后进入MySQL Installer 页面选择 Custom并点击Next

  • 通过在项目栏里的→选中MySQL Server 8.0.44 - X64后点击 Next

  • 在 Download部分点击 Execute进行数据库的下载，下载完成后点击Next进入 Installation页面

  • Installation页面点击 Execute进行数据库的安装，安装完成后点击Next进入Production Configuration页面后再次点击Next进入Installation Complete页面的配置页面

  • 勾选 Show Advanced ..后点击Next进入下一个页面继续点击Next进入Accounts and Roles

  • 输入root用户密码（建议使用123456方便记忆）并点击Next进入下一个页面

  • 修改服务名称为MySQL后继续点击Next进入下一个页面选择No .....后继续点击Next进入下一个页面

  • 继续Next

  • 继续Next

  • 继续NExt

  • 点击Execute进行最后的配置后点击Finish完成最后的安装

8.Navicat安装

  • 点击并打开安装包navicat17_premium_lite_cs_x64.exe

  • 点击下一步后选择我同意后继续点击下一步

  • 点击浏览选择安装地址为D:\Navicat Premium Lite 17点击下一步

  • 继续点击下一步后点击安装

  • 安装成功后打开Navicat Premium Lite 17,注册并登录后进入软件

  • 一直点击下一步跳过新功能通知，选择不共享

  • 进入软件后点击左上角的连接

  • 选择MySQL点击下一步后输入刚刚安装mysql的密码(123456或者是你另外定下来的)

  • 点击左下角测试连接，如果下方提示连接成功，则点击确定完成连接

  • 双击左侧的MySQL即可连接数据库

（二）、系统环境变量的设置与安装验证

1.系统环境变量设置

  • 点击 Win 键后输入 查看高级系统设置并打开 系统属性页面

  • 点击 环境变量 并进入下方页面

  • 点击 系统中的 新建，并在 新建系统变量页面填写如下 变量名与变量值后点击确定

  • 在 系统变量 中下拉并寻找 Path 这一栏并选中双击打开 编辑环境变量 页面

  • 新建以下变量

    • %JAVA_PATH%\bin

    • D:\WorkEnv\Maven\maven-mvnd-1.0.3-windows-amd64\maven-mvnd-1.0.3-windows-amd64\bin

    • D:\WorkEnv\Maven\maven-mvnd-1.0.3-windows-amd64\maven-mvnd-1.0.3-windows-amd64\mvn\bin

  • 新建好变量后点击 确定

  • 环境变量 页面再次点击 确定

  • 系统属性页面再次点击 确定 按钮

  • 系统环境变量配置到这里就结束了

2.安装环境的验证

  • 键盘键入 Win + I 组合键，在弹出的 运行 窗口输入 cmd并确定后打开命令提示符窗口

  • 开始按照以下顺序进行安装的验证

    • 输入 java --version,如果出现下方类似版本号则JAVA SDK的安装及环境变量配置成功！

    • 输入 mvn -v，如果出现下方类似版本号则Maven的安装及环境变量配置成功

    • 输入 node -v,如果出现下方类似的版本号则NodeJs的安装及环境变量配置成功

    • 输入 git -v,如果出现下方类似的版本号则Git的安装及环境变量配置成功

四、项目搭建

Ⅰ.使用IDEA创建项目后端环境

  • 打开 IDEA并且点击 新建项目

  • 新建项目 页面在左侧选择生成器 Spring Boot并且按照下图填写内容

  • 点击JDK 选择 从磁盘添加JDK，并输入路径 D:\WorkEnv\Java并点击确认

  • Java选择 25

  • 在确定你的项目名称以及Java JDK配置和管理包选择Maven之后点击下一步

  • 在新建项目页面勾选以下5个选项并点击创建后耐心等待项目环境创建

  • 当右下角的下载任务完成并且出现以下文件结构则说明项目后端环境创建成功

  • 打开终端并输入 mvn spring-boot:run,执行命令后如果终端内容末尾有类似于日志记录的内容以及则代表项目后端成功搭建并且已经启动

后端项目启动命令： mvn spring-boot:run

Ⅱ.使用VsCode创建项目前端环境

  • 按下Win键并输入git bash后选择 以管理员身份运行

  • 运行以下命令进行Vue的安装

    • 输入并运行 npm install vue -g等待安装

    • 输入并运行 npm install @vue/cli -g

    • 输入并运行 npm install vue-router -g

  • 打开 VsCode并且打开刚刚创建的后端项目文件夹

  • 键盘键入Ctrl + J组合键打开终端，点击图片中展示的 ⅴ然后选择 Git Bash并点击打开如下终端页面
  • 输入并运行vue create frontend

  • 输入 y使用官方镜像源

  • 键入Enter选择默认的Vue3并耐心等待vue初始化

  • 输入并运行 cd frontend && npm run serve并耐心等待项目前端启动

  • 出现下方内容则说明前端成功启动

Ⅲ.前后端启动命令

  • 前端启动命令

    cd frontend && npm run serve

  • 后端启动命令

    mvn spring-boot:run
